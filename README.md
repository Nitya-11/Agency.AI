# 🚀 Digital Agency Website

A modern, fully responsive Digital Agency website built with React, Tailwind CSS, and Framer Motion. The project features smooth animations, dark/light theme support, responsive layouts, and a functional contact form powered by Web3Forms.

## 🌐 Live Demo



## 📂 GitHub Repository

https://github.com/Nitya-11/Agency.AI

---

## ✨ Features

- Responsive design for all devices
- Modern UI with Tailwind CSS
- Smooth animations using Framer Motion
- Dark & Light mode toggle
- Animated custom cursor
- Sticky navigation
- Smooth scrolling
- Interactive service cards
- Portfolio showcase section
- Team section
- Functional Contact Form using Web3Forms
- Clean and reusable React components

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Framer Motion
- Web3Forms
- React Icons

## 🔑 Environment Variables

Create a `.env` file in the root directory.

```env
VITE_WEB3FORMS_ACCESS_KEY=YOUR_ACCESS_KEY
```

---

## 📬 Contact Form

The contact form is integrated with **Web3Forms**, allowing visitors to send messages directly to the administrator's email without requiring a custom backend.
